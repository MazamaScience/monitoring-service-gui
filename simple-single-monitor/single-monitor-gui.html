Monitor Plots





Instructions

  1. Select which type of plot you would like to generate.
  2. Type in the ID of the monitor you would like to inspect. You can find a monitor's ID by clicking on its icon in this map and the ID will appear in the "Selected Monitors" list at the bottom of the page.
  3. Select the date range by clicking on the date widget. You can type in the start and end dates or select a start and end day in the calendar popup.

(Changing any of the fields will automatically generate a plot. The image may take a few seconds to load)
